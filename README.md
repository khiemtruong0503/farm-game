# Farm Game Project

## Introduction

The Farm Game is an engaging and educational farming simulation developed using Unity 2D. Players manage a farm by planting crops, feeding animals, customizing layouts, and trading goods in the market. The game aims to provide a realistic farming experience while being accessible and enjoyable for players of all ages.

## Features

- **Farm Management**: Plant, grow, and harvest various crops.
- **Animal Care**: Feed animals, monitor their health, and collect products. (not finished)
- **Customization**: Personalize your farm layout and structures.
- **Market System**: Buy and sell crops and animals in a dynamic market.
- **Weather System**: Real-time weather effects that influence farming activities. (not finished)
- **Progression**: Gain experience, level up, and unlock new items.

## Installation

1. Clone the repository:
    ```sh
    git clone https://github.com/yourusername/farm-game.git
    ```
2. Open the project in Unity 2D:
    - Ensure you have Unity Hub installed.
    - Add the project to Unity Hub and open it with Unity 2D.

3. Install dependencies:
    - Follow any additional setup instructions provided in the `Dependencies` section of the repository.

## Game Mechanics

- **Planting Crops**: Select crop types, prepare the soil, plant seeds, water them, and wait for them to grow through different stages until harvest.
- **Harvesting**: Harvest fully grown crops and add them to your inventory.
- **Customizing Layout**: Use drag-and-drop functionality to arrange fields, barns, and other structures on your farm.
- **Market**: Buy and sell resources to optimize your farm's productivity.
- **Weather System**: Adapt to changing weather conditions that affect crop growth and animal health.
